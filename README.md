### 🎮 Playground Mini-Apps
Six UI experiments that run live inside the portfolio — click any card to open:

| App | What it does | Techniques |
|---|---|---|
| **Animated Login** | Rotating orange bar ring + glassmorphism card | CSS keyframes, Vanilla JS |
| **Sliding Auth Form** | Login/register with panel slide transition | CSS transitions, Form UX |
| **Futuristic Terminal** | Sci-fi ACCESS login with grid + glitch effect | CSS animations, SVG filters |
| **3D Electric Cards** | Hover-tilt cards with turbulence border glow | SVG feTurbulence, 3D CSS |
| **Neon Auth** | Neon-bordered login/register with shape animation | CSS transform, clip-path |
| **Card Slider** | Fullscreen hero carousel with text reveal | DOM manipulation, CSS animation |

## 🚀 Deploy

### Option 1 — Vercel (recommended)
1. Fork or clone this repo
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import this repo
4. Set **Root Directory** to `/` and **Output Directory** to `.`
5. Deploy ✅

### Option 2 — GitHub Pages
1. Go to repo **Settings → Pages**
2. Set source to **main branch / root**
3. Your site goes live at `https://Ezenri-Elvis-CodElv.github.io/portfolio`

### Option 3 — Run locally
```bash
# Just open the file — no server needed
open index.html
# or on Windows:
start index.html
```
